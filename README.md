# QuietPlaces – zemljevid mirnih mest

## Opis aplikacije
Aplikacija predstavlja zemljevid mirnih mest, ki jih označujejo drugi uporabniki. 
Vsako mesto ima svoj opis in natančno lokacijo na zemljevidu.
Uporabniki prav tako prejemajo potisna obvestila o novih mestih v bližini.

## Namen aplikacije
Omogočiti uporabnikom, da vidijo mirna mesta na zemljevidu za učenje, počitek ali drugo spokojno preživljanje časa.
